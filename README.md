# Izdevumu izsekotājs

Vienkāršs Python/tkinter lietojums personīgo izdevumu uzskaitei.

## Funkcijas
* Izdevumu pievienošana (summa, kategorija, apraksts).
* Datu saglabāšana JSON failā starp sesijām.
* Mēneša filtrs un automātiska kopsummas aprēķināšana.
* Datu eksportēšana uz CSV failu priekš Excel.

## Uzstādīšana
1. Pārliecināties, ka ir instalēts Python 3.10+.
2. Lejupielādējiet visus failus (`app.py`, `logic.py`, `storage.py`, `export.py`).

## Lietošana
1. Palaidiet programmu ar komandu: `python app.py`.
2. Ievadiet izdevuma datus un nospiediet "Pievienot".
3. Izmantojiet dropdown izvēlni, lai filtrētu datus pēc mēneša.
4. Nospiediet "Eksportēt uz CSV", lai saglabātu datus tabulas formātā.
